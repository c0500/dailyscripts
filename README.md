# dailyscripts
scripts to make daily tasks easy

- addsshauth.sh
 add rsa pub key for a remote ssh host, so you can login remote hosts without password.
 usage:
 ```
 addsshauth.sh user@host
 ```

- bypassvpn.sh
 by pass vpn for specfic hosts, means route to default gateway.
 usage:
 ```
 sudo bypassvpn.sh
 ```
